# SpectroVirtu CIS Website

Responsive Vite + React presentation website for the SpectroVirtu Corporate Identity System.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

- `src/App.jsx` contains the CIS presentation sections.
- `src/styles.css` contains the full responsive visual system.
- `assets/spectrovirtu-logo-2026.svg` is the provided logo and is used without redesign.
